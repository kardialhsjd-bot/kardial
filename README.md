# KARDIAL – Gestión de Informes Cardiológicos

Aplicación web profesional para gestión de informes de electrocardiograma (ECG).

## Características

- 📋 Generación de informes ECG desde archivos PDF/imagen
- 👥 Gestión de pacientes y base de datos
- 🔑 Sistema de roles y autenticación (Admin, Enfermería, TENS, Doctor)
- 📊 Estadísticas y métricas de actividad
- 📥 Exportación a PDF e impresión
- ✍️ Firma digital de informes

## Tecnología

- HTML5 + CSS3 + JavaScript Vanilla
- PDF.js para visualización de documentos
- html2pdf.js para exportación PDF

## Despliegue

Desplegado automáticamente en Vercel desde el repositorio principal.
